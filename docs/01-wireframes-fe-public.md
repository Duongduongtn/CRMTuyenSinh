# Wireframe FE Public — CRM Tuyển Sinh Học Lái Xe

Phiên bản: v1 draft, ngày 10/06/2026.
Đối tượng: người Việt Nam 25-55 tuổi, ưu tiên trung niên cần học lái xe lần đầu hoặc nâng hạng.
Mục tiêu: landing SEO, conversion cao, vibe trust-first.

> **Quy tắc thiết kế cốt lõi (rút từ taste-skill):**
> - DESIGN_VARIANCE = 6, MOTION_INTENSITY = 5, VISUAL_DENSITY = 4
> - Palette: deep navy (text) + cream (bg) + 1 accent emerald hoặc amber
> - Font: Geist Sans (hỗ trợ dấu tiếng Việt)
> - KHÔNG em-dash (`—`), KHÔNG 3 card đều, KHÔNG fake screenshot, KHÔNG AI purple
> - Asymmetric hero, bento hoặc zig-zag features, ảnh thật

**Schema JSON-LD nhúng trong head:**
- `LocalBusiness` (DrivingSchool) với address, phone, openingHours, geo.
- `BreadcrumbList` (chỉ Home).
- `FAQPage` cho section FAQ.
- `AggregateRating` nếu thật sự có Google reviews.

**Motion gợi ý:**
- Hero: ảnh fade-in từ opacity 0 + translateY 24px, 600ms cubic-bezier(0.16, 1, 0.3, 1).
- Khóa học bento: stagger 80ms khi vào viewport (Motion `whileInView`).
- Đánh giá carousel: auto-play 5s mỗi slide, pause on hover.
- CTA buttons: scale 0.98 on press, không magnetic.

## Responsive

| Breakpoint | Đặc điểm |
|---|---|
| Mobile (< 768px) | Hero stack dọc, ảnh full width trên, text dưới. Bento chuyển 1 cột. Sidebar khóa chi tiết đẩy xuống dưới content. Modal full screen. |
| Tablet (768-1024px) | Bento 2 cột. Sidebar khóa chi tiết vẫn dọc nhưng width hẹp hơn. |
| Desktop (> 1024px) | Bento 3 cột + ô lớn. Sidebar sticky. |

## Accessibility

- Contrast text/bg ≥ WCAG AA 4.5:1.
- Focus ring visible trên mọi interactive element.
- Aria labels cho icon buttons.
- Skip-to-content link đầu trang.
- `prefers-reduced-motion`: disable scroll animation, giữ fade nhẹ.
- Form label luôn ở trên input, error message dưới input có aria-live.

## SEO checklist

- Mỗi trang có `<title>` riêng, max 60 ký tự.
- `<meta description>` riêng, max 160 ký tự.
- OG image 1200x630 riêng cho mỗi loại trang.
- Schema JSON-LD đã liệt kê từng trang.
- Sitemap XML auto-generate từ DB.
- robots.txt cho phép crawler.
- URL slug có dấu tiếng Việt đã thoát (vd: `khoa-hoc/b-so-san`).
- Core Web Vitals: LCP < 2.5s (ảnh hero lazy + priority), CLS < 0.1, INP < 200ms.
